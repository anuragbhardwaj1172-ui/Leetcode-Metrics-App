# LeetMetric App 🚀

**LeetMetric** is a simple and stylish web application that tracks and displays real-time coding statistics and problem-solving progress for any LeetCode user. By fetching data directly from the LeetCode API, the application showcases user statistics using dynamic progress circles and clean, organized cards.

---

## ✨ Features

* **Real-time Data Fetching:** Utilizes the LeetCode GraphQL API to fetch live, up-to-date user statistics[cite: 2].
* **Dynamic Progress Circles:** Visually represents the ratio of solved questions for Easy, Medium, and Hard difficulties using custom dynamic circular gradients[cite: 1, 2].
* **Stats Cards:** Displays total overall submissions and difficulty-wise submissions in a clean, responsive grid layout[cite: 1, 2].
* **Modern UI:** Built with a clean, responsive dark-themed interface using pure HTML and CSS Flexbox[cite: 1, 3].

---

## 🛠️ Tech Stack Used

* **HTML5:** For structuring the application layout and UI elements.
* **CSS3:** For custom styling, the dark-mode aesthetic, circular progress indicators, and flexbox responsiveness[cite: 1].
* **JavaScript (ES6):** For username regex validation, asynchronous API handling (Async/Await), and dynamic DOM manipulation[cite: 2].

---

## 📂 File Structure

* `html_project.html` — The main landing page containing the user interface structure[cite: 3].
* `css_project.css` — The stylesheet containing layout definitions and design components[cite: 1].
